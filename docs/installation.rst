============
Installation
============


Stable release
--------------

To install Set Your Lineup Reminder, run this command in your terminal:

.. code-block:: console

    $ pip install set_your_lineup_reminder

This is the preferred method to install Set Your Lineup Reminder, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for Set Your Lineup Reminder can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/RandyAkerman/set_your_lineup_reminder

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/RandyAkerman/set_your_lineup_reminder/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/RandyAkerman/set_your_lineup_reminder
.. _tarball: https://github.com/RandyAkerman/set_your_lineup_reminder/tarball/master
